彼岸图库爬虫
2021年05月05日 318 字 大概 1 分钟

1.首先导入需要的数据库.

import requests
import re
import time

2.这里是获取你要爬取哪一页的图片,需要填写.

if name == ‘main‘:
page = input(‘选择你爬取的页数2-132’)
url = f’http://www.netbian.com/dongman/index_{page}.htm'
get = requests.get(url).content.decode(‘gbk’)

3.创造一个循环数量等于单页内图片的数量.

a = 0
while a < 19:
    a = a + 1

4.获取总网页中的子网页中的图片网页.(说的有点乱,但话糙理不糙.)

    div = re.findall('<a href="(.*?)" title="(.*?)" target="_blank">', get)[a][0]
    idurl = f'http://www.netbian.com{div}'
    picurl = requests.get(idurl).content.decode('gbk')
    pic = re.findall('<img src="(.*?)" alt="(.*?)" title="(.*?)">', picurl)[0][0]
    name = re.findall('<img src="(.*?)" alt="(.*?)" title="(.*?)">', picurl)[0][-1]

5.打印出图片的名称和图片的链接并打印爬取成功的信息. 

    print(name)
    print(pic)
    print('爬取成功''\n')

6.保存图片,路径按自己需求,需要有此文件夹.

    img = requests.get(pic)
    time.sleep(0.3)
    with open(f'(！！路径！！)/{page}----，第{a}张--,{name}.jpg', 'wb') as girl:
        girl.write(img.content)

7.保存后的结果就是 (页数)—-第(几)张–(图片名字).jpg
